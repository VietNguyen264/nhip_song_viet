
# Dự án web tĩnh chủ đề 'Khám phá Việt Nam'
## Mục đích của project nhằm học cách vận dụng 2 thư viện 'Boostrap 5' và 'jQuery' 
- Ngôn ngữ: HTML, Css, Javascript
- IDE: Visual Studio Code  
- Giới thiệu project: Áp dụng thư viện framework Boostrap 5 để tạo animation cho các thành phần Css trong web và jQuery cho các thành phần Javascript
- Thành phần chính của project:
  - 7 trang html: index.html (trang chủ), event.html (trang sự kiện), place.html (trang địa điểm), food.html (trang ẩm thực), contact.html (trang tương tác và liên hệ,
    sign-in.html (giao diện đăng nhập người dùng), sign-up (giao diện đăng ký tài khoản) ;
  - 1 file style.css dùng chung cho toàn bộ html ; 
  - 1 file main.js thiết lập tính năng.
